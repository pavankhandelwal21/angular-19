• To use Route tag add File in app.Route
  • Import RouterLink and RouterOutlet in component file
  • User router link in tags and make sure you are adding "router-outlet/" in the html file to open the details in link
  • To add the color to active line use "routerLinkActive="active"" in the tag where we used routerLink
  • If we are in home and it doesnt has any link make sure you use '[routerLinkActiveOptions]="exact: true"' this will check the exact link
